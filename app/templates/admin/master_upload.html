How It Works

  1. Upload your Head_Count_MAA.xlsx and Head_Count_BLR.xlsx files above.
  2. The app reads the HeadCount_Base sheet from each file.
  3. Every row with a User ID is saved to the agent_master table.
  4. On every roster view, agent_master data takes priority over BigQuery data for: WIN ID, Full Name, LOB/Team, Role, L1 Manager, L2 Manager.
  5. The roster team filter dropdown also uses clean LOB names from the master data.
  6. Override Editor and Ad-Hoc Roster also use enriched names and teams.
